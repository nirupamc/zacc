Troubleshooting

Rate Limiting Issues

If you get "too many requests" errors with Spotify URLs:

  • Set up Spotify API authentication
  • Try smaller playlists (10-20 tracks)
  • Wait 5-10 minutes between downloads
  • Use YouTube playlist URLs instead

Playlist Not Found

If you get "404" or "not found" errors:

  • Make sure the playlist is public
  • Check the URL is copied correctly
  • Try opening the URL in your browser first
  • Some private playlists may not be accessible

Network Issues

If downloads fail or timeout:

  • Check your internet connection
  • Try a VPN if services are blocked
  • Retry smaller playlists first
  • Contact support if issues persist
